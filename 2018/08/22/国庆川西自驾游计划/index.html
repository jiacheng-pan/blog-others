国庆川西自家有计划

去程：杭州-成都（飞机）

07:45-10:45，预计出机场11:45，取车、吃饭、采购完毕12:30（最迟13:00）

第一天：成都市-卧龙-四姑娘山

  • G350/G317：231公里，路程时间4.5小时，游玩时间1.5小时。

1. 成都-卧龙大熊猫苑：2小时

  • 蓉昌高速+G350：115公里，2小时。

2. 卧龙中华大熊猫苑神树坪基地：1小时（视情况而定）

  • 进山费15。

  • 卧龙中华大熊猫苑神树坪基地：门票90，在游客中心买票，可以开车到景区门口。电瓶车自愿，没必要。

  • 在此游玩一小时。

3. -四姑娘山猫鼻梁：2小时

  • G350：94公里，2小时。

  • 在此观赏落日，半小时。

4. 猫鼻梁-四姑娘山住宿：半小时

  • G350：15公里，半小时

  • 在此住宿

第二天：四姑娘山-甲居藏寨-观音桥镇

  • G350/S211：312公里，路程7.5小时，游览时间4小时

1. 四姑娘山双桥沟：3小时

  • 门票80，观光车70

2. 四姑娘山双桥沟-甲居藏寨：3.5小时（136公里）

3. 甲居藏寨：1小时

在甲居藏寨游玩1小时，甲居藏寨是中国最美六大乡村古镇。

4. 甲居藏寨-观音桥镇：4小时（179公里）

5. 观音桥镇

  • 住宿在观音桥镇

第三天：观音桥镇-色达五明佛学院-色达县/翁达镇

  • G317：200公里-250公里，路程3.5小时-4.5小时

（这一段路视情况而定，如果色达县城没有住宿了，可以考虑住在翁达镇）

1. 观音桥镇：1小时

  • 参观观音桥镇、观音庙

2. 观音桥镇-色达五明佛学院：3小时

  • G317（成那线）：183公里，3小时

3. 色达五明佛学院：直至夜色降临

  • 天葬下午1点-2点开始

4. 色达五明佛学院-色达县：0.5小时（首选）

  • V18县道：20公里，夜晚行驶预计0.5-1小时

4’. 色达五明佛学院佛学院-翁达镇：1小时

  • G317(成那线)：62公里，夜晚行驶预计1-2小时

第四天：色达县/翁达镇-新龙县-理塘县

  • 这段路比较危险，县级公路，路况很不好，要慢慢开，即便饿肚子，也要安全第一。
  • 翁达出发：400公里，路程7.5小时
  • 色达出发：390公里，路程11小时

1. 色达县-新龙县：8小时

  • G317+S217：238公里

  • 路况较差，路比较近，开车较慢，但景观自然

1’. 翁达镇-新龙县：4.5小时

  • G317+S217：245公里
  • 翁达至甘孜，路况较好；剩余路就比较一般。

2. 新龙县+措卡湖：视情况而定

  • 如果时间充裕，可以前往措卡湖

3. 新龙县-理塘县：3小时

  • S217：152公里
  • 这段路路况较差，况且今年刚有事故，一定小心
  • 这一段路慢慢到达理塘县，海拔超过4000米，要注意高反

第五天：理塘县-新都桥镇-折多山-海螺沟

1. 理塘县-新都桥镇：4小时

  • G318：195公里

2. 新都桥：时间视情况而定

  • 在新都桥镇歇脚吃饭
  • 号称“摄影天堂”
  • 十里长廊

3. 新都桥-折多山观雪台：1小时

  • G318：42公里

4. 折多山观雪台（逗留半小时）

  • 折多山的盘山公路九曲十八弯，注意驾驶安全，折多山比较拼人品，天气不好就是条路。。。

5. 折多山-海螺沟（磨西古镇）：3.5小时

  • G317：108公里

6. 磨西古镇+海螺沟

  • 在磨西古镇吃饭后，入住海螺沟景区，第二天起来看日出

第六天：海螺沟-成都

1. 海螺沟：玩到不想玩为止

  • 票价：90
  • 海螺沟国家森林公园以其大型低海拔的现代冰川著称于世，冰川长达14.7公里，风光壮观。 景区融原始森林、珍稀动植物、温泉、瀑布于一体，可近距离接触冰湖、冰洞、冰裂隙、冰阶梯等多种景观。 游玩海螺沟的主要目的是泡温泉（二号营地）和看冰川（三号营地）。 二号营地有海螺沟内名气最大的热水沟沸泉，目前已建有10个大小不等的露天温泉池和一个的露天温泉游泳池。
  • Tips: 入口设在磨西镇，购票进入后，便可乘坐观光车前行。 景区观光车无强制必买，但整个行程几十公里，不坐观光车基本不可能。观光车发车时间07:30-13:00，12:50停止售票，坐满一车发一班，间隔不超过40分钟发车。 游览海螺沟最好在冬春季节，夏秋季冰川会因为温度缘故有一定回缩，且夏秋季阴雨天较多，不便于观看日照金山景观。 各营地海拔及间距：磨西镇（海拔1600米）-15公里-—号营地（海拔1940米）-8公里-二号营地（海拔2660米）-7.5公里-三号营地（海拔2980米）-3.5公里-索道站-四号营地（海拔3600米）

2. 海螺沟-成都：5小时

  • 雅康高速、成渝高速：300公里

第七天：成都吃吃吃
